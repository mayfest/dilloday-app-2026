## setup (first time ppl)
**Mac**
1. Open XCode
2. Open a random project
3. Press `Cmd + ,` to open Settings
4. Go to the Platforms (or Components) tab
5. Ensure IOS 26.2 is installed if not already
6. Go to VS Code and open up this repo and open temrinal
7. `npx expo run:ios`
NOTE: You will get an error at first when you try to run it (at least for me). Simply re-run the above command, or press 'i' to activate IOS within the terminal.
